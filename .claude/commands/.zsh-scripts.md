# .zsh-scripts

Show all shell scripts in your dotfiles bin directory.

## Instructions

1. List all executable scripts in `~/code/dotfiles/bin/`
2. For each script, extract:
   - Script name
   - First comment/description from the file (if available)
   - Whether it's executable
   - Brief purpose (from filename or header comments)

3. Group by category:
   - **Tmux**: tmux-*.sh scripts
   - **Aerospace**: aerospace-*.sh scripts
   - **Vault/Genie**: vault, genie-related scripts
   - **Git**: git-related utilities
   - **Other**: miscellaneous utilities

4. Use Glob to find: `~/code/dotfiles/bin/*.sh` and `~/code/dotfiles/bin/*` (executables)
